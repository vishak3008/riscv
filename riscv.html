Browser-Based RISC Playground

Build math instructions and watch a RISC processor execute them live.

This project simulates a compact RISC-style CPU in JavaScript. Type assembly-like instructions, run them cycle by cycle, and inspect registers, memory, ALU results, and the program counter.
